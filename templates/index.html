About This Demo

This application demonstrates semantic music search using pgvector with Render's PostgreSQL. It uses sentence transformers to convert natural language queries into vector embeddings, then performs similarity search against a database of song embeddings.

Technology Stack

  • Backend: Python Flask
  • Database: PostgreSQL with pgvector
  • ML Model: SentenceTransformer (paraphrase-MiniLM-L3-v2)
  • Deployment: Render
  • Data: Kaggle music dataset

Key Features

  • Natural language music search
  • Semantic similarity matching
  • Real-time recommendations
  • YouTube & Spotify integration
  • Responsive design
